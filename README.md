# Phishing Website Detection Using Machine Learning

## Project Overview

This project focuses on detecting phishing websites using machine learning techniques.

Phishing websites are a major cybersecurity threat because they attempt to imitate legitimate websites to steal sensitive information. This project applies a classification model to identify patterns in website characteristics that can help distinguish phishing websites from legitimate websites.

The project follows the CRISP-DM (Cross Industry Standard Process for Data Mining) framework.

---

# Problem Statement

The goal of this project is to build a machine learning model that can classify websites as phishing or legitimate based on website-related features.

The model aims to support cybersecurity analysis by identifying suspicious website patterns.
