
# Project Reflection

This project helped me understand how machine learning can be applied to cybersecurity problems such as phishing website detection.

I learned the importance of data exploration, preparation, model training, and evaluation.

## Challenges Faced

The main challenges were understanding the dataset structure, preparing the data, and selecting a suitable machine learning model within the available time.

## How the Challenge Was Solved

I solved these challenges by exploring the dataset, cleaning and preparing the data, applying feature scaling, and using Logistic Regression for classification.
